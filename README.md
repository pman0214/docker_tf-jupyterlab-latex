# TensorFlow + Jupyterlab + LaTeX

## Overview

* Docker image based on [pman0214/tf_jupyterlab](https://hub.docker.com/r/pman0214/tf_jupyterlab).
* Installs LaTeX for EPS plot output.
* Also provides sample `docker-compose.yml` files.
